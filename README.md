# Delivery Management System

A Delivery Management System with Admin, Customer, and Rider modules built using C# Windows Forms, with file handling for data storage.

## Modules & Features

### Admin
- Login system
- User Management (manage all customers and riders)
- Manage Delivery Riders
- Order Management
- Reports & Analytics (revenue, statistics)
- View Rates

### Rider
- Login system
- View Assigned Deliveries
- View Optimized Routes
- Analytics Dashboard

### Customer
- Login system
- Track Order Status (Delivered, In Progress, etc.)
- View Orders & Location

## Tech Stack

- C#
- Windows Forms
- File Handling

## How to Run

1. Make sure you have **Visual Studio** installed (2019 or later)
2. Clone or download this repository
3. Open `DSAproject.sln` in Visual Studio
4. Press **F5** or click **Start** to build and run
